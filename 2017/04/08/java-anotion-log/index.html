介绍如何使用java注解来添加日志。

注解

元注解

元注解的作用是注解其他注解，

  • @Target用于描述注解范围。

  • @Retention用于描述注解生命周期。

  • @Documented用于生命构建注解文档。

  • @Inherited用于描述该注解是可被子类继承的。
