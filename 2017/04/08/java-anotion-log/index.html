Java 日志注解

介绍如何使用Java注解来添加日志。

注解

元注解

元注解的作用是注解其他注解，

  • @Target用于描述注解范围。

  • @Retention用于描述注解生命周期。

  • @Documented用于生命构建注解文档。

  • @Inherited用于描述该注解是可被子类继承的。

注解格式

日志注解

Log注解定义，参数包含type日志类型、desc日志描述、throwable是否catch异常、withResult是否记录打印结果。

需要注意，注解的默认值必须是常量（不可以设置为枚举）
