golang项目结构与其他语言类似，但是仍然有一些需要注意的地方。

项目结构

环境配置

go 命令依赖一个重要的环境变量：$GOPATH，它表示GO项目的路径，如下设置

对于GOPATH来说，允许多个项目目录（Unix中为“：”，Windows中为“；”）。

项目目录

在项目目录中，一般包含三个文件夹，分别为src，pkg和 bin。各个文件夹功能如下，

  • src 存放golang源码
  • pkg 存放编译后的文件
  • bin 存放编译后可执行的文件

模块导入

在golang中，模块导入包括两种导入方式：相对路径和绝对路径。
